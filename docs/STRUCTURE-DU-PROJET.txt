STRUCTURE PRINCIPALE

index.html              : page publique des projets
apropos.html            : présentation, diplômes et compétences
readme.html             : affichage local des README
projects-config.json    : choix publiés depuis le dashboard
admin/dashboard.html    : outil privé de gestion
assets/                 : CSS, JavaScript, images et polices
docs/                   : notices et documentation du projet
Projet_qui_ne_sont_pas_en_lien/ : pages locales de présentation

Le code du dashboard n'est plus écrit en clair. Le fichier dashboard-access.js
contient uniquement une empreinte SHA-256 et limite les tentatives dans l'onglet.

data/ : listes générées utilisées par le dashboard
outils/ : scripts locaux pour lancer le site et actualiser la liste des pages

PAGES LOCALES :
- Une page qui correspond à un dépôt GitHub est fusionnée avec ce dépôt.
- Une page sans dépôt correspondant reste proposée dans le dashboard.
